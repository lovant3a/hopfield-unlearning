# «Сон» и раз-обучение в аттракторных нейронных сетях

Курсовая работа по теме: **подавление ложных аттракторов и улучшение восстановления памяти в сети Хопфилда**

## Описание

Классическая сеть Хопфилда реализует ассоциативную память, но при высокой загрузке возникают спурионные (ложные) аттракторы, ухудшающие качество восстановления. В проекте реализована процедура **раз-обучения** («сна») по Kleinfeld (1986), которая ослабляет связи, поддерживающие ложные минимумы энергии.

## Структура проекта
├── run_experiments.py # Запуск экспериментов и сохранение результатов
├── plot_results.py # Построение графиков
├── requirements.txt # Зависимости
└── README.md

## Установка и запуск

**Требуется Python 3.10+**

```bash
# 1. Клонирование репозитория
git clone https://github.com/USERNAME/REPO_NAME.git
cd REPO_NAME

# 2. Установка зависимостей
pip install -r requirements.txt

# 3. Запуск вычислений (~3-5 минут)
python run_experiments.py

# 4. Построение графиков
python plot_results.py

После выполнения в папке figs/ появятся графики:
rec_vs_alpha.png — качество восстановления vs загрузка сети
spur_vs_alpha.png — частота спурионных состояний
unlearning_heatmap.png — зависимость от гиперпараметров
Параметры по умолчанию
Размер сети: N = 100
Загрузка: α = P/N ∈ [0.05, 0.18]
Шум при тестировании: 10% битов
Число испытаний: 1000 на точку
Seed: 42 (для воспроизводимости)
